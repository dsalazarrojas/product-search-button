Product Search Button

Support

Find a product faster across the stores and search services you already use.

What it does

Product Search Button adds a search button to product cards on supported online stores. Select a search destination to manually look up the product on Google, Perplexity, supermarkets, marketplaces, department stores, or deal sites.

Install and use

  1. Install Product Search Button from your browser’s extension store.
  2. Open a supported online store and browse product cards.
  3. Select the Product Search Button on a product card.
  4. Choose a search destination. The selected third-party website opens with a query based on the visible product information.

Configure the extension

Open the extension’s settings/options page from your browser’s Extensions menu. There you can:

  • Set priority stores: choose and order the store destinations shown first in the search menu.
  • Choose the menu corner: select where the product-card button/menu appears.
  • Choose submenu direction: set whether the submenu opens to the left or right to fit the page layout.
  • Choose a default search engine: select which destination is used as the default option.

A store search or selector is broken

Store websites change their product-card markup from time to time. If the button is missing, appears in the wrong place, or creates an incorrect query, please email contact@gic.mx.

Include: the store name and page URL, what you expected, what happened, your browser, and—if possible—a screenshot with personal information removed.

Please do not send passwords, payment information, or other sensitive personal information in a support request.
